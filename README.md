# 測一測你是哪一種台灣麵？

一個以「台灣復古麵店」為風格的單頁心理測驗網頁遊戲。玩家回答 10 題單選題後，系統會依照選項累積分數，判斷玩家最接近哪一種台灣麵人格。

## 遊戲結果

測驗共有四種人格結果：

- 汕頭麵：直接、豪爽、外冷內熱、重朋友。
- 鹽水意麵：溫柔、細膩、念舊、有生活感。
- 油麵：效率高、實際、理性、適應力強。
- 刀削麵：有韌性、抗壓強、獨立、有個性。

## 計分規則

測驗共有 10 題，每題 4 個選項，固定對應如下：

- A = 汕頭麵
- B = 鹽水意麵
- C = 油麵
- D = 刀削麵

最後統計四種麵的分數，分數最高者就是結果。如果最高分同分，會從同分結果中隨機選出一個。

## 介面功能

- 開始頁
- 題目頁
- 題目進度顯示，例如 `第 3 / 10 題`
- 動態進度條
- 點選答案後自動進入下一題
- 結果頁人格診斷卡
- 重新測驗按鈕
- 題目切換與結果淡入動畫
- 手機與桌面響應式版面

## 使用方式

這是純前端專案，不需要後端或資料庫。

最簡單的方式是直接用瀏覽器開啟：

```text
index.html
```

也可以用本機伺服器執行，例如：

```bash
python -m http.server 8000
```

然後在瀏覽器開啟：

```text
http://127.0.0.1:8000
```

## 設計方向

整體視覺採用米白背景、紅色招牌、木質卡片與菜單式結果卡，營造台灣復古麵店感。字體使用偏圓潤與手寫感的中文字體搭配，避免制式模板感。
